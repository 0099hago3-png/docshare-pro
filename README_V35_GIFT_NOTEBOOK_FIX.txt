DOCSHARE PRO V35 - SỬA TẶNG QUÀ + SỔ TAY NỔI BẬT + KHOẢNG CÁCH CHỮ

Đã sửa:
1. Sửa phần tặng quà để dùng lại ổn định.
2. Chia quà thành các nhóm: quà khởi đầu / quà ủng hộ / quà vinh danh.
3. Mỗi món quà sau khi tặng có hiệu ứng riêng theo chủ đề.
4. Thêm khu "Quà vừa được tặng" trong trang chi tiết tài liệu.
5. Sửa "Sổ tay tài liệu nổi bật" để bấm được.
   - Nguyên nhân cũ: khung library-scene bị pointer-events:none.
6. Tăng khoảng cách chữ với khung ở phần tiêu đề / hero.
7. Đổi STORAGE_KEY để tránh cache localStorage cũ.

Nếu vẫn thấy bản cũ:
- Ctrl + F5
- Hoặc DevTools -> Application -> Local Storage -> xóa dữ liệu localhost.
