# DocShare Pro V28 – Green Academic Library

## Nội dung chính
- Đồng bộ các mảng tối cũ sang xanh lục thư viện.
- Sửa lỗi Admin trắng trang do biến chỉ mục trong biểu đồ.
- Ví, lịch sử, rút tiền, kho quà, top ủng hộ và bảng xếp hạng dùng màu xanh lục dễ đọc.
- Thu nhỏ avatar về kích thước gọn, khung vẫn ôm sát.
- Trang chủ có sổ tài liệu nổi bật, lật bằng nút mũi tên và chấm trang.
- Làm lại mini chat với màu xanh – kem, chữ rõ.
- Chat hỏi đáp có 5 nhóm và 27 câu hỏi mẫu có thể bấm chọn, kèm trả lời chi tiết.
- Thêm lá cây trang trí nhẹ ở các nền và thẻ lớn.

## Kiểm tra
```bash
npm ci
npm run build
```
