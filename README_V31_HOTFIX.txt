DOCSHARE PRO V31 - HOTFIX BẢNG XẾP HẠNG

Đã sửa lỗi còn sót trong ảnh người dùng gửi:
- Tên Top 1, Top 2, Top 3 chuyển sang màu xanh đen đậm.
- Dòng cấp độ/người theo dõi chuyển sang xám xanh đậm.
- Số điểm chuyển sang màu tối, không còn chữ trắng chìm trên nền sáng.
- Tên phần thưởng chuyển sang nâu vàng đậm.
- Giữ riêng chữ trong avatar và số thứ hạng màu trắng để dễ nhìn.
- Gỡ nền xanh đậm bao quanh cụm Top 3, thay bằng từng thẻ kem/xanh nhạt riêng.

Sau khi giải nén:
1. Chạy npm install
2. Chạy npm run dev
3. Nhấn Ctrl + F5

Nếu tab localhost cũ đang mở, hãy đóng tab đó rồi mở lại để chắc chắn Vite tải CSS mới.
