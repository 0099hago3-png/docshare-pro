# DocShare Complete V24

## Thay đổi chính

- Đổi toàn bộ website sang chủ đề thư viện học thuật màu trắng, kem, xanh Oxford và vàng đồng.
- Bỏ toàn bộ thú cưng và khu túi thú cưng khỏi giao diện.
- Bỏ hiệu ứng chuyển động/phát sáng mạnh; giao diện tĩnh, rõ ràng và chuyên nghiệp.
- Làm lại logo DocShare thành biểu tượng sách có chữ D.
- Làm lại bìa tài liệu theo kiểu bìa sách chuyên nghiệp.
- Làm lại toàn bộ khung avatar theo kiểu viền mảnh, tĩnh, màu học thuật.
- Chỉ giữ Túi khung avatar; không hiển thị danh hiệu và túi bảng.
- Sửa tác giả ở trang chi tiết tài liệu thành liên kết có thể bấm để xem hồ sơ.
- Tối ưu bố cục desktop theo cảm giác 90%, giảm độ phóng và tăng khoảng trắng.
- Thêm bố cục riêng cho tablet và điện thoại.
- Giữ nguyên các chức năng V23: xem trước, mua, tải, bình luận, đánh giá, báo cáo, lịch sử và upload.

## Cài đặt

```bash
npm install
npm run dev
```

## Kiểm tra production

```bash
npm run build
```
