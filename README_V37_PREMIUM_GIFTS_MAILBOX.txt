DOCSHARE PRO V37 - QUÀ CAO CẤP + HIỆU ỨNG MỚI + HỘP THƯ USER/ADMIN

Đã sửa và thêm mới:

1. Quà cao cấp hơn
- Tăng số quà lên 27 món.
- Thêm quà cao cấp mới:
  - Siêu Tân Tinh Tri Ân
  - Du Thuyền Học Giả
  - Vương Miện Ngọc Lục Bảo
  - Tinh Vân Kim Cương
  - Lễ Hội Pháo Hoa Tri Thức

2. Hiệu ứng quà khác biệt hơn
- Quà cấp Legendary có hiệu ứng mạnh hơn.
- Nền, ánh sáng, hạt bay, vòng sáng được tăng cấp rõ hơn.
- Bảng quà tặng vẫn hiển thị hiệu ứng riêng theo chủ đề.

3. Bảng quà tặng kiểu cửa sổ
- Modal quà được làm lại theo kiểu cửa sổ.
- Có header giống cửa sổ desktop.
- Khung nhìn gọn, rõ, dễ bấm.

4. Hộp thư cho người dùng và admin
- Trang Tin nhắn được làm lại thành Hộp thư.
- Người dùng có thể:
  - xem các thư đã có
  - lọc thư với admin
  - soạn thư mới
  - gửi thư cho admin / tác giả
- Admin Dashboard có thêm tab "Hộp thư".
- Admin có thể đọc và trả lời thư trực tiếp.

5. Đã kiểm tra build
- npm install: OK
- vite build: OK

Lưu ý sau khi thay file:
- npm install
- npm run dev
- Ctrl + F5
- Nếu còn dữ liệu cũ: DevTools -> Application -> Local Storage -> xóa localhost
