DOCSHARE PRO V34 - ĐÃ LÀM LẠI CARD TÀI LIỆU

Lý do sửa:
- Bản trước vẫn bị CSS cũ đè nên card tài liệu bị hẹp, cao và lỗi bố cục.

Đã sửa trong V34:
1. Làm lại component card tài liệu bằng layout mới hoàn toàn.
2. Card giống kiểu ảnh mẫu hơn: bìa ở trên, thông tin ở dưới.
3. Tỷ lệ bìa thống nhất, không còn bị kéo thành cột hẹp.
4. Tài liệu nổi bật / Gợi ý cho bạn / Top yêu thích dùng card ngang cuộn như mẫu.
5. Vẫn giữ:
   - hiện credit
   - hiện tích xanh
   - hiện miễn phí / đã sở hữu / cần credit
6. Đổi STORAGE_KEY để xóa cache giao diện cũ trong localStorage.

Nếu vẫn thấy bản cũ:
- Ctrl + F5
- Hoặc vào DevTools -> Application -> Local Storage -> xóa localhost.
